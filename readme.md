#deadbybot

## required library:

python-dotenv
discord.py

## how to set up the environment :

- create bot-env

  - python3 -m venv bot-env

- activate virtual env:

  - source bot-env/bin/activate

- install in env
  - pip install -U discord.py
  - pip install -U python-dotenv

## how to run the bot :

- activate virtual env if not already in it:
  - source bot-env/bin/activate
- run bot:
  - python3 main.py
