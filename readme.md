# dbd_bot

## how to set up the environment:

- create env:
  python3 -m venv env

- activate virtual env:
  source env/bin/activate

- install requirements:
  pip install -r requirements.txt

- activate virtual env if not already in it:
  source env/bin/activate

## how to run the bot:

- python3 main.py

## env vars :

- bot_token
- api_url
